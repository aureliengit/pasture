Note importante
---------------

Penser à utiliser l'URL complète pour la feuille de style :

  <!-- Bootstrap -->
  <link href="http://squidguard.nestor/css/bootstrap.min.css" rel="stylesheet">
  <link rel="stylesheet" href="http://squidguard.nestor/css/open-sans.css" type="text/css" />
  
De même pour les images :

  <p><a href="http://httpd.apache.org/"><img
  src="http://squidguard.nestor/images/apache_pb.gif" alt="[ Powered by Apache
  ]"></a> <a href="http://www.centos.org/"><img
  src="http://squidguard.nestor/images/poweredby.png" alt="[ Powered by CentOS
  Linux ]" height="31" width="88"></a></p>
